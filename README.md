# Fintech Pulse — Transaction Intelligence Analysis

Analysis of 2,512 banking transactions to uncover customer behaviour 
patterns, fraud signals and product recommendations.

## Key Findings
- Students (18-30) are the highest spenders per transaction
- 77% of transactions are Debits — current account behaviour
- Branch dominates channels — digitally underserved customers
- Suspicious login patterns identified — potential fraud signal
- Middle aged customers hold most wealth but spend least

## Tools
Python, Pandas, Matplotlib, Jupyter Notebook

## Dataset
Kaggle — Bank Transaction Dataset for Fraud Detection
